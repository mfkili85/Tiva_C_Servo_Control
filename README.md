"# Tiva_C_Servo_Control" 
Servo PWR -> Tiva_C VBUS
Servo GND -> Tiva_c GND
Servo signal -> Tiva_C PDO

use Tiva C push buttons to control Servo angle position:
  - Left button set servo angle = 0 deg.
  - right button set servo angle to 90 deg.
